# オセロゲーム

シンプルなJavaScript製のオセロゲームです。

## 遊び方

1.  `index.html` ファイルをブラウザで開きます。
2.  黒の番から始まります。
3.  相手の石を挟める位置に自分の石を置くことができます。
4.  石を置くと、挟んだ相手の石が自分の色に変わります。
5.  盤面がすべて埋まるか、両者とも置ける場所がなくなるとゲーム終了です。
6.  最終的に石の数が多い方の勝ちです。

## ファイル構成

-   `index.html`: ゲームのメインファイル
-   `style.css`: スタイルシート
-   `script.js`: ゲームのロジック
